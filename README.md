🚀 CuddleCode – Real-time Multiplayer Coding Platform

CuddleCode is a modern, full-stack real-time coding battle platform where users can compete, chat, and solve coding problems together in a live environment.

✨ Features
⚔️ Real-time multiplayer coding battles
💬 Live chat between players
⏱️ Synchronized timers for fair competition
🧠 Multi-language code execution (C++, Java, Python, JavaScript)
📝 Monaco-powered in-browser code editor
🏆 Leaderboard & matchmaking system
🎉 Animated UI with victory effects
📸 Shareable result screenshots
🛠️ Tech Stack
🎨 Frontend
React (Vite)
Framer Motion
Monaco Editor
Socket.io-client
Lucide React
Canvas APIs (canvas-confetti, html2canvas)
Vanilla CSS
⚙️ Backend
Node.js
Express.js
Socket.io
💻 Execution Engine
Node.js vm (JavaScript sandboxing)
child_process (C++, Java, Python execution)
💾 Storage
File-based JSON system (users, leaderboard, problems)
⚙️ How It Works
1.Users join a battle room
2.Problems are assigned in real-time
3.Code is written in Monaco Editor
4.Code is sent to backend for execution
5.Execution engine compiles/runs code against test cases
6.Results are returned instantly and leaderboard updates live
