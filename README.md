🚀 CuddleCode – Real-Time Coding Battle Platform

CuddleCode is a modern, real-time multiplayer coding platform where users can compete head-to-head by solving coding challenges in multiple programming languages.

🌟 Features
⚔️ Real-time multiplayer coding battles
💬 Live chat using WebSockets
🧠 Multi-language code execution (JavaScript, Python, C++, Java)
🔐 Secure authentication (JWT + bcrypt)
🏆 Leaderboard & XP system
⚡ Instant result evaluation with hidden test cases
🎨 Smooth UI with animations and modern design
🛠️ Tech Stack

🎨 Frontend
1.React + Vite
2.Framer Motion
3.Monaco Editor
4.Socket.io-client

⚙️ Backend
1.Node.js + Express
2.Socket.io
3.MongoDB + Mongoose
4.JWT Authentication

💻 Execution Engine
1.Node.js VM (JavaScript sandbox)
2.Child Process (Python, C++, Java execution)
3.Time & memory constrained execution

🔐 Security Features
1.Sandboxed code execution
2.Execution time limits
3.Rate limiting & HTTP security (Helmet)
4.Input validation & sanitization

🌐 Deployment
Backend hosted on Render
Frontend hosted on Vercel

Project Structure
client/   → Frontend (React)
server/   → Backend (Node.js + Express)

Future Improvements:-
1.Match history tracking
2.Player stats dashboard
3.Ranked matchmaking system
4.Docker-based execution sandbox
